# How To Use CAT

Select a city, set walking time and walking speed, choose a start point, enable the comfort variables that matter to the user, set how much each one affects their comfort, and run the catchment area analysis.

The agent should frame the calculation around the environmental comfort factors the user cares about and how strongly each factor affects them. Pre-set profiles are only shortcuts for quickly applying typical settings; they must not be presented as if one profile can represent every individual. The user should be able to review walking speed, walking time, enabled variables, their impact levels, unavailable variables, and the optional pre-set source before analysis runs.
