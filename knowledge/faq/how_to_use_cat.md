# How To Use CAT

Select a city, set walking time and walking speed, choose a start point, enable comfort variables, adjust their sensitivity, and run the catchment area analysis.

The agent can help by detecting a user profile from natural language and proposing CAT settings. The user should be able to review the profile, walking speed, walking time, enabled variables, unavailable variables, and action before analysis runs.
