# CAT Troubleshooting

If no reachable area appears, check that a start point is selected, walking time and walking speed are reasonable, and at least one supported city dataset is available.

If comfort-adjusted area is empty, reduce the sensitivity of selected discomfort variables or try fewer variables.

If a requested variable is not applied, it may not be available for the selected city.
