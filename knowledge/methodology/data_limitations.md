# Data Limitations

Data availability differs by city. A variable available in Hamburg may not be available in Penteli.

RAG knowledge can explain available datasets and limitations, but it cannot provide local spatial truth for a selected point or catchment area.

The frontend and APIs must show missing data warnings when requested or preset variables are filtered out.
