# Result Interpretation

Result explanations must use actual result metadata, such as default reachable area, comfort-adjusted reachable area, comfort ratio, enabled variables, POI count, and missing data warnings.

The agent may explain that a smaller comfort-adjusted area suggests that selected comfort factors reduce comfortable walking accessibility.

The agent must not claim local physical causes, such as many stairs or poor pavement, unless that fact was computed or retrieved from an approved spatial tool.
