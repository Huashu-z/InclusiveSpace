# Wheelchair User Profile

Profile name: Wheelchair user.

Typical walking speed: 3.0 km/h.

Relevant comfort variables: stair, kerbsHigh, slope, narrowRoads, poorPavement, unevenSurface, obstacle, wcDisabled, trafficLight.

Recommended variable values:
- stair: 0.1
- kerbsHigh: 0.2
- slope: 0.3
- narrowRoads: 0.3
- poorPavement: 0.3
- unevenSurface: 0.3

Explanation: Wheelchair users are strongly affected by stairs, high kerbs, steep slopes, narrow sidewalks, damaged pavement, uneven surfaces, and obstacles. Accessible toilets and safe crossings can support independent movement.

Warnings or limitations: The profile should not assume ramps or barrier-free routes unless those facts are available from actual CAT data or result metadata.
